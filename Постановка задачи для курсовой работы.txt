Курсовая работа 

Тема: Создание БД для хранения и последующей обработки данных из системы управления проектами (Jira).

Потребность: Рабочая группа осуществляет планирование и фиксирует результаты своей деятельности в рамках отдельной системы управления проектами/задачами (Jira).
Базовый функционал формирования отчетности не позволяет проводить детальный анализ эффективности работы команды, а также не позволяет формировать отчеты для руководства в соответствии с принятым в компании стандартом.

Верхнеуровневый план работ:
1. Подготовить инфраструктуру для обеспечения хранения данных из системы источника (Jira).
2. Настроить ETL-процессы по передаче данных из системы-источника в требуемое хранилище.
3. Разработать на базе копоративного BI-решения дашборды для ежедневного мониторинга деятельности рабочей группы, а также для предоставления итоговой отчетности за выбранный период времени.

В рамках курсововой работы будет реализован п.1. из плана выше.

Доп. требования к решениею:
1. СУБД - PostgreSQL 10
2. Какую информацию необходимо хранить:
	1. Общая информация о зарегистрированных задачах
	2. Справочник пользователей
	3. История изменений по каждой задаче
	4. Данные для учета рабочего времени, затраченного на реализацию задачи
	5. Справочник нормативов (норма рабочего времени)
	6. Данные по выпущенным релизам
	7. Комментарии к задачами
	8. Справочник для группировки по направлениям работ
	9. Данные о присвоенных метках
	10. Данные о зависимостях между задачами
